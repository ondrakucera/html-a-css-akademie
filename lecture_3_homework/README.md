# Domácí úkol z 3. lekce

Dokonči úlohu z výuky. Připrav HTML strukturu úvodní stránky novin (jejichž jednotlivými články jsou povídky z _Povídání
o pejskovi a kočičce_), a to za použití elementů _nav_, _main_, _aside_ a _article_ (pochopitelně kromě jiných).

Stránku zatím není potřeba nijak stylovat (to bude obsahem závěrečné lekce).
